# UANGIN - Aplikasi Pengelolaan Keuangan

Aplikasi web untuk mengelola pemasukan dan pengeluaran harian dengan fitur budgeting dan reporting.

## Dokumentasi

Lihat [README lengkap](docs/README.md) untuk informasi instalasi, penggunaan, dan pengembangan.

## Struktur Proyek

- `backend/` - Server Express.js dengan API
- `frontend/` - Halaman web statis dengan JavaScript
- `database/` - Schema dan data SQL
- `scripts/` - Script setup dan maintenance
- `docs/` - Dokumentasi

## Quick Start

1. Install dependencies: `npm install`
2. Setup database: `node scripts/init-db.js`
3. Start server: `npm start`

Untuk detail lengkap, lihat [dokumentasi](docs/README.md).
